## JavaScript 编程

### 实现深拷贝

```javascript
const deepClone = (obj, hash = new WeakMap()) => {
  // 非 object 类型直接返回
  if (obj === null || typeof obj !== "object") return obj;

  // 如果 hash 里已经存在，则返回
  if (hash.has(obj)) {
    return hash.get(obj);
  }

  // 初始化新对象，数组特殊处理
  const cloneObj = Array.isArray(obj) ? [] : {};

  // 存入 hash
  hash.set(obj, cloneObj);

  for (let key in obj) {
    cloneObj[key] = deepClone(obj[key], hash);
  }

  return cloneObj;
};

const obj = { name: "Tom" };
obj.self = obj;

const copy = deepClone(obj);
console.log(copy);
```

使用 [`WeakMap`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/WeakMap) 作为 hash 表，避免循环引用导致栈溢出。

使用 `WeakMap` 而不使用 `Map`，是因为 `WeakMap` 的键是弱引用，不会阻止垃圾回收器回收被克隆的对象，从而避免内存泄漏。

现成的深拷贝可以使用 [`structuredClone`](https://developer.mozilla.org/en-US/docs/Web/API/Window/structuredClone)。
