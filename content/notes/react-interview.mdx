## History and evolution of React

2011 Birth of React

2013 Open sourcing React

2015 Flux & state management

> Flus is an architectural pattern

2016 Functional components & Fiber architecture

> Fiber is React's core architecture

2018 React hooks

React 16.8 introduced Hooks (`useState`, `useEffect`, etc.)

2020 Concurrent mode & Suspense

> Concurrent allowing React to prioritize rendering work React Suspense made it
> easier to handle data fetching

2021 React Forget

> It's a compiler that automatically generates the equivalent of `useMemo` and
> useCallback calls to minimize the cost of re-rendering.

2022 React 18 and Server Components

> Server Components allowing developers to offload rendering work to the server.

2024 React 19

> optimized Server Components
> React Compiler was announced
